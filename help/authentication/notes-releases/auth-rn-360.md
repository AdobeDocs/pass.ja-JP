### リリースの概要 {#release-overview-360}

このリリースでは、Adobe Pass認証サービス全体の品質、安定性、およびセキュリティ更新プログラムに焦点を当てます。

#### 機能強化

* REST API V2の信頼性、識別可能性、応答一貫性に関する一般的な改善。
* 認証フローとリクエスト処理の強化。
* 一般的なシナリオでの互換性を向上させるために、パートナーとMVPDの統合動作を改善しました。
