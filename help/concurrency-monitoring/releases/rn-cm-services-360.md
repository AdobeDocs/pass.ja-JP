## リリース情報 {#rel-info}

バージョン：3.6.0
日付：2025 年 2 月 4 日（PT）

## リリースの概要 {#release-overview}

* API を更新して、他のテナントのアプリで実行されているストリームの詳細を含めます。
* API メソッドに終了コードを追加します。
* 409 応答および変更可能なメタデータ属性のメタデータを追加して、メタデータ API を更新しました。
